важно переделать spa на ssr

1) семантические теги - нормально улучшает ранджированность
2) тег title и open graph желательно на каждой странице свой - сильно ранджированность
3) тег h1 на каждой странице свой - сильно улучшает ранджированность
4) человеко понятные ссылки, никаких чисел - немного улучшает ранджированность 
5) избавиться от дубликатов (canonical links) - не тратит круленговый бюджет
6) настроить через генератор robots.txt файл - позволяет не ранджированность страницы админки 
7) настроить sitemap файл - позволяет точно дать понять какие ссылки присутствуют на сайте
